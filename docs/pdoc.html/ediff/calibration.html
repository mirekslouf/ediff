ediff.calibration

Module: ediff.calibration

The calibration of electron diffraction patterns:

  • The original diffraction pattern shows intensities as a function of distance-in-pixels.
  • The calibrated diffraction pattern shows intensities as a function of distance-in-q-vectors.
  • The term distance-in-q-vectors means that the distances = magnitudes of q-vector in [1/A].
  • This module gives the calibration constant, which converts distance-in-pixels to distance-in-q-vectors.
  • The final conversion is very simple: distance_in_q = distance_in_pixels * calibration_constant

How to determine the calibration_constant:

  • In EDIFF, the calibration constant is calculated in ediff.calibration.Calculate
  • The calculation is simplified for known microscopes in ediff.calibration.Microscopes
  • Links to real-life examples are given at GitHub pages: https://mirekslouf.github.io/ediff/docs

 69    def from_max_peaks(eld_profile, xrd_profile, messages=True):
 70        '''
 71        Calibration constant from the *maximal* peaks on ELD and PXRD profiles.
 72
 73        * ELD/XRD profiles represent 1D electron/X-ray diffraction patterns.
 74        * More info about ELD/XRD profiles in EDIFF
 75          => see the docs of ediff.io.read_profile function.
 76
 77        Parameters
 78        ----------
 79        eld_profile : str or numpy.array
 80            The *eld_profile* (ELD) is
 81            an electron diffraction profile in EDIFF format.
 82            It can come as file (if *eld_profile* = str = filename)
 83            or array (if *eld_profile* = numpy.array).
 84        xrd_profile : str or numpy.array
 85            The *xrd_profile* (XRD) is
 86            an X-rayd diffraction profile in EDIFF format.
 87            It can come as file (if *xrd_profile* = str = filename)
 88            or array (if *xrd_profile* = numpy.array).
 89        messages : bool, optional, default is True
 90            If *messages* = True,
 91            print some information
 92            and the final calibration constant to stdout.
 93        
 94        Returns
 95        -------
 96        calibration constant : float
 97            The multiplicative constant that converts
 98            ED-profile X-coordinate-in-pixels
 99            to X-coordinate-in-q-vectors [1/A].        
100        '''
101        
102        # Function {Calculate.from_max_peak} is a special case
103        # of more general function {Calculate.from_max_peaks_in_range},
104        # which we call without specifying the ranges => to use whole ranges.
105        calibration_constant = Calibrate.from_max_peaks_in_range(
106            eld_profile, xrd_profile, messages=messages)
107        
108        # Return calibration constant
109        return(calibration_constant)

112    def from_max_peaks_in_range(
113            eld_profile, xrd_profile,
114            eld_range=None, xrd_range=None, messages=True):
115        '''
116        Calibration constant from the *selected* peaks on ED and PXRD profiles.
117        
118        * ELD/XRD profiles represent 1D electron/X-ray diffraction patterns.
119        * More info about ELD/XRD profiles in EDIFF
120          => see the docs of the ediff.io.Profile.read function.
121        * The peaks are selected using arguments *eld_range* and *xrd_range*.
122            - Both arguments are tuples of two floats = x-ranges.
123            - Only the maximal peaks in given ranges are considered.
124            - ED range is in [pixels] and PXRD range is given in [q-vectors].
125
126        Parameters
127        ----------
128        eld_profile : str or numpy.array
129            The *eld_profile* is
130            an electron diffraction profile in EDIFF format.
131            It can come as file (if *eld_profile* = str = filename)
132            or array (if *eld_profile* = numpy.array).
133        xrd_profile : str or numpy.array
134            The *xrd_profile* is
135            an X-rayd diffraction profile in EDIFF format.
136            It can come as file (if *xrd_profile* = str = filename)
137            or array (if *xrd_profile* = numpy.array).
138        eld_range : tuple of two floats, optional, default is None
139            The x-range in 1D ED profile,
140            in which we should search for the maximal peak.
141            The ED x-range is given in [pixels].
142        xrd_range : tuple of two floats, optional, default is None
143            The x-range in 1D XRD profile,
144            in which we should search for the maximal peak.
145            The XRD x-range is given in [q-vectors].
146        messages : bool, optional, default is True
147            If *messages* = True,
148            print some information
149            and the final calibration constant to stdout.
150            
151        Returns
152        -------
153        calibration constant : float
154            The multiplicative constant that converts
155            ED-profile X-coordinate-in-pixels
156            to X-coordinate-in-q-vectors [1/A].
157        '''
158        
159        # (1) Read ED and XRD diffraction profiles.
160        # ELD and XRD are ediff.io.Diffractogram1D objects.
161        # TODO: more general reading ?
162        eld = eld_profile
163        xrd = xrd_profile
164            
165        # (2) Determine ranges, in which we search for peaks.
166        # (We search peaks either in the whole x-ranges
167        # (or in sub-ranges, if the args eld_range and xrd_range are given.
168        # ! X-ranges for ED and XRD are given in [pixel] and [q], respectively.
169        if eld_range is not None:
170            emin,emax = eld_range
171            eld = eld[ (emin <= eld.Pixels) & (eld.Pixels <= emax)]
172        if xrd_range is not None:
173            xmin,xmax = xrd_range
174            xrd = xrd[ (xmin <= xrd.q) & (xrd.q <= xmax) ]
175        
176        # (3) Get the peak/maximum value for ED and XRD.
177        # (The ranges where to search for peaks were determined in prev.step.
178        max_eld = float( eld.loc[eld.I.idxmax(), 'Pixels'] )
179        max_xrd = float( xrd.loc[xrd.I.idxmax(), 'q'] )
180        if messages is True:
181            print(f'Max.peak in ELD-profile at {max_eld:6.4f} [Pixels]')
182            print(f'Max.peak in XRD-profile at {max_xrd:6.4f} [q(1/A)]')
183        
184        # (4) Calculate calibration constant
185        # (the constant converts d[pixels] to q[1/A]
186        calibration_constant = max_xrd/max_eld
187        if messages is True:
188            print(f'Calibration constant: {calibration_constant:.6f}')
189        
190        # (5) Return the calculated calibration constant
191        return(calibration_constant)

194    def from_microscope(microscope, voltage_kV=None, 
195        camera_length_mm=None, nominal_camera_length_mm=None,
196        camera_pixel_size_um=None, binning=None, verbose=False):
197        '''
198        Calibration constant from known {microscope+camera} system.
199         
200        * Known microscopes are in ediff.calibration.Microscopes class.
201        * Each known microscope contains info about four key parameters:
202          voltage_kV, camera_length_mm, camera_pixel_size_um, and binning.
203        * These parameters can be (optionally) modified when calling this func.
204        * This function collects all four arguments,
205          either from the microscope object or from the optional arguments,
206          and then it calls ediff.calibration.Calibrate.from_microscope_params.
207        
208        Parameters
209        ----------
210        microscope : str or microscope object
211            One of the microscopes defined in ediff.calibration.Microscopes.
212            We can use either a string with the sub-class name
213            (such as: 'TecnaiVeleta')
214            or a full subclass name
215            (such as: Microscopes.TecnaiVeleta).
216        voltage_kV : int, optional, default is None
217            Accelerating voltage in [kV].
218            If {None}, the value is taken from {microscope} argument.
219        camera_length_mm : float, optional, default is None
220            *Real* camera lenght in [mm].
221            If {None}, the value is taken from {microscope} argument.
222            IMPORTANT: see the {Note on camera length} section below!
223        nominal_camera_length_mm : float, optional, default is None
224            *Nominal* camera lenght in [mm].
225            If {None}, the *real* camera lenght 
226            is calculated from {microscope} argument.
227            IMPORTANT: see the {Note on camera length} section below!
228        camera_pixel_size_um : float, optional, default is None
229            Camera pixel size in [um].
230            If {None}, the value is taken from {microscope} argument.
231        binning : int
232            Binning=1,2,4,... increases camera_pixel_size (1x,2x,4x,...).
233            If {None}, the value is taken from {microscope} argument.
234        verbose : bool, optional, default is False
235            If {True}, the function prints outputs to stdnout.
236             
237        Returns
238        -------
239        calibration constant : float
240            The multiplicative constant that converts
241            (for given electron diffraction experiment/pattern)
242            X-coordinate-in-pixels to X-coordinate-in-q-vectors [1/A].
243             
244        Note on camera length
245        ---------------------
246        * The {camera_length_mm} argument can be:
247            - *nominal camera length* (D) shown by the microscope SW
248            - *real camera length* = real CL, usually different from D
249        * The *nominal camera length* (D) may be different from
250          the *real camera length* (CL) because:
251            - the difference in the position of top x bottom camera
252            - projective lenses (which may change the mag. of diff. pattern)
253        * Conclusions:
254            - For function `Calibrate.from_microscope`,
255              which employs a calibrated microscope from Microscopes class,
256              we can insert both {camera_length_mm} = real camera length
257              or {nominal_camera_length_mm} = nominal camera lenght.
258              calibrated microscope has the constant converting
259              *nominal* to *real* camera length.
260            - For function `Calibrate.from_microscope_params`,
261              which employs an arbitrary/non-calibrated microscope
262              (i.e. a microscope that is not defined in Microscopes class)
263              we have to insert the *real camera length*
264              in order to get the correct result.
265        '''
266        # (0) If the {microscope} name was given as a string,
267        # convert it to the full subclass name.
268        if isinstance(microscope, str):
269            microscope = Microscopes.str_to_class_name(microscope)
270       
271        # (1) Take the params from microscope class if they were not specified.
272        # ... 1st parameter = accelerating voltage
273        if voltage_kV is None: 
274            voltage_kV = microscope.voltage_kV
275        # ... 2nd parameter = camera_length    
276        if (camera_length_mm is None) and (nominal_camera_length_mm is None):
277            camera_length_mm = microscope.D * microscope.D_to_CL_coefficient
278        elif camera_length_mm is not None:
279            pass  # the camera_length_mm was defined as argument, Ok
280        elif nominal_camera_length_mm is not None:
281            camera_length_mm = \
282                nominal_camera_length_mm * microscope.D_to_CL_coefficient
283        # ... 3rd parameter = camera pixel size
284        if camera_pixel_size_um is None:
285            camera_pixel_size_um = microscope.camera_pixel_size_um
286        # ... 4th parameter = binning
287        if binning is None:
288            binning = microscope.binning
289        
290        # (2) Call Calibrate.from_microscope_params funct
291        # (with all arguments collected in the previous step
292        calibration_constant = Calibrate.from_microscope_params(
293            voltage_kV, camera_length_mm,
294            camera_pixel_size_um, binning,
295            verbose)
296        
297        # (3) Return the calculated calibration constant
298        return(calibration_constant)

301    def from_microscope_params(
302        voltage_kV, camera_length_mm, 
303        camera_pixel_size_um, binning, verbose=False):
304        '''
305        Calibration constant from arbitrary {microscope+camera} parameters.
306        
307        * The calibration constant can be estimated from parameters
308          of given microscope + camera system.
309        * The parameters we need to know are: (i) accelerating_voltage,
310          (ii) camera_length, (iii) camera_pixel_size, and (iv) binning.
311        
312        Parameters
313        ----------
314        voltage_kV : float
315            Accelerating voltage in [kV].
316            If {None}, the value is taken from {microscope} argument.
317        camera_length_mm : float
318            Camera lenght in [mm].
319            If {None}, the value is taken from {microscope} argument.
320            IMPORTANT: see the {Note on camera length} section below!
321        camera_pixel_size_um : float
322            Camera pixel size in [um].
323            If {None}, the value is taken from {microscope} argument.
324        binning : int
325            Binning=1,2,4,... increases camera_pixel_size (1x,2x,4x,...).
326            If {None}, the value is taken from {microscope} argument.
327        verbose : bool, optional, default is False
328            If {True}, the function prints outputs to stdnout.
329
330        Returns
331        -------
332        calibration constant : float
333            The multiplicative constant that converts
334            (for given electron diffraction experiment/pattern)
335            X-coordinate-in-pixels to X-coordinate-in-q-vectors [1/A].
336        
337        Note on camera length
338        ---------------------
339        * The {camera_length_mm} argument can be:
340            - *nominal camera length* (D) shown by the microscope SW
341            - *real camera length* = real CL, usually different from D
342        * The *nominal camera length* (D) may be different from
343          the *real camera length* (CL) because:
344            - the difference in the position of top x bottom camera
345            - projective lenses (which may change the mag. of diff. pattern)
346        * Conclusions:
347            - For function `Calibrate.from_microscope`,
348              which employs pre-defined microscopes from the
349              ediff.calibration.Microscopes class
350              we insert the *nominal camera length*, because we know
351              the conversion coefficient from nominal to real camera length.
352            - For function `Calibrate.from_microscope_params`,
353              which employs arbitrary microscope
354              (i.e. microscope that is not defined in the Microscopes class)
355              we have to insert the *real camera length*
356              in order to get the correct result.
357        '''
358                
359        # (1) Calculate relativistic wavelength of electrons
360        # (this is needed to convert CL to CC in the next step
361        Lambda = Utils.electron_wavelength(voltage_kV)
362        
363        # (2) Determine CL and CC
364        # * CL = CameraLenght
365        #   CL should be known from exp => argument camera_length_mm
366        # * CC = Camera Constant
367        #   CC = CL * Lamda => CC[mmA] = CL[mm]*Lamda[A]
368        #   The CC-CL => Camera equation: R*d = Lambda*CL = CC
369        CL = camera_length_mm
370        CC = camera_length_mm * Lambda
371        
372        # (3) Calculate final calibration constants.
373        # * The final calibration constants are:
374        #   R_calibration = pixel size in [mm]
375        #   S_calibration = k_calibration = pixel size in S [1/A]
376        #   q_calibration ............... = pixel size in q = 2*pi*S [1/A]
377        # * To calculate all three constants, we need just two parameters:
378        #   camera_pixel_size  = real size of the camera pixels
379        #   CC = camera_length = camera length, determined above
380        # * The calculation is easy
381        #   but it is performed in a separate function,
382        #   because it is employed also in calibration.Microscopes class.
383        # * The function contains
384        #   more details, including full justification of the calculations.
385        R_calibration, S_calibration, q_calibration = \
386            Utils.calc_final_calibration_constants(
387                camera_pixel_size_um = camera_pixel_size_um,
388                binning = binning,
389                camera_constant_mmA = CC)
390        
391        # (4) Print the calculated values if requested
392        if verbose:
393            Utils.print_final_calibration_constants(
394                Lambda, CL, CC,
395                R_calibration, S_calibration, q_calibration)
396
397        # (5) Return the calculated calibration constant
398        # (We want the calibration constant in q [1/A]
399        calibration_constant = q_calibration
400        return(calibration_constant)

403class Microscopes:
404    '''
405    A dataclass containing known/calibrated microscopes.
406
407    * The microscopes are defined as subclasses.
408    * More precisely, the microscopes are sub-dataclasses.

412    >>> import ediff as ed
413    >>>
414    >>> # (1) Define the microscope + its parameters
415    >>> # (a) Simple case - all parameters at their default/standard values
416    >>> my_TEM = ed.calibration.Microscopes.TecnaiVeleta()
417    >>> # (b) Intermediate case - standard values + camera lenght changed

420    >>> # (c) The most comlex case - multiple parameters have changed

422    >>>     nominal_camera_length_mm=750, voltage_kV=80, binning=4)
423    >>>
424    >>> # (2) Calculate the calibration constant
425    >>> const = ed.calibration.Calculate.from_microscope_parameters(my_TEM)

431    * Example: {Tecnai microscope + Veleta camera}

433    * The final calculation of the calibration constant is made by the

585        '''
586        Calculate relativistic wavelenght of accelerated electrons.
587    
588        Parameters
589        ----------
590        U : float
591            Accelerating voltage [kV].
592    
593        Returns
594        -------
595        Lambda : float
596            Relativistic wavelenght of electrons,
597            which were accelerated with the voltage *U*[kV]
598            in an electron microscope.
599        '''
600        # The formula below gives Lambda = wavelength in [A]
601        # (justification: textbooks of physics
602        # (my source: M:/MIREK.PRE/0_EM.PY/1_ELN-IN-EM/1eln_v-wl_v2.nb.html
603        
604        # Convert U from [kV] to [V]
605        U = U * 1000
606        
607        # Collect constants from scipy
608        h = scipy.constants.h
609        c = scipy.constants.c
610        e = scipy.constants.e 
611        m = scipy.constants.m_e
612        
613        # Calculate and return relativistic wavelenght of electrons
614        Lambda = h/np.sqrt(2*m*e*U) * (1/(np.sqrt(1+(e*U)/(2*m*c**2)))) * 1e10
615        return(float(Lambda))

618    def calc_final_calibration_constants(
619            camera_pixel_size_um, binning, camera_constant_mmA):
620        '''
621        Calculate final calibration constants
622        from known CC and camera_pixel_size.
623
624        * Once we know *CC*, *camera_pixel_size*, and *binning*,
625          the calculation of the final calibration constants
626          is surprisingly easy.
627        * The full justification of the calculation
628          is given in the comments of the source code below.
629        
630        Parameters
631        ----------
632        camera_constant_mmA : float
633            The camera constant (from R*d = CL*Lambda = CC) in [mmA].
634        camera_pixel_size_um : float
635            The real dimension of one pixel of the camera (detector) in [um].
636        binning : int
637            Binning = 1,2,4,... increases pixel size (1x,2x,4x,...)
638
639        Returns
640        -------
641        R_calibration, S_calibration, q_calibration : three floats
642            * R_calibration = the camera pixel size in [mm].
643            * S_calibration = the camera pixel size in S-vector units [1/A].
644            * q_calibration = the camaera pixel size in q-vector units [1/A].
645        '''
646
647        # What do we want to calculate?
648        #  => three calibration constants = pix_size, S-units, q-units        
649        # Brief justification:
650        #  (a) Camera Equation: R*d = CL*Lambda = CC
651        #  (b) Bragg's Law: 2*d*sin(theta) = Lamda => S*d = 1 => S = 1/d
652        #  (c) Combine (a)+(b) for a known R[um] => here: R = size of 1 pixel
653        #      in general : R*d = CC => S = 1/d = R/CC
654        #      here/below : S_calibration[1/A] = R_calibration[mm] / CC [mm*A]
655        # -----
656        # Note: camera_pixel_size is influenced by binning
657        
658        # Calculate the calibration constants acc.to justification above
659        R_calibration = camera_pixel_size_um * binning / 1000
660        S_calibration = R_calibration / camera_constant_mmA
661        q_calibration = 2*np.pi * S_calibration
662
663        # Return the three calibration constants
664        return(R_calibration, S_calibration, q_calibration)

134        xrd_profile : str or numpy.array
135            The *xrd_profile* is

137            It can come as file (if *xrd_profile* = str = filename)
138            or array (if *xrd_profile* = numpy.array).
139        eld_range : tuple of two floats, optional, default is None
140            The x-range in 1D ED profile,
141            in which we should search for the maximal peak.
142            The ED x-range is given in [pixels].
143        xrd_range : tuple of two floats, optional, default is None
144            The x-range in 1D XRD profile,

160        # (1) Read ED and XRD diffraction profiles.
161        # ELD and XRD are ediff.io.Diffractogram1D objects.
162        # TODO: more general reading ?
163        eld = eld_profile
164        xrd = xrd_profile
165            
166        # (2) Determine ranges, in which we search for peaks.
167        # (We search peaks either in the whole x-ranges
168        # (or in sub-ranges, if the args eld_range and xrd_range are given.
169        # ! X-ranges for ED and XRD are given in [pixel] and [q], respectively.
170        if eld_range is not None:

172            eld = eld[ (emin <= eld.Pixels) & (eld.Pixels <= emax)]

188        if messages is True:

190

619    def calc_final_calibration_constants(
620            camera_pixel_size_um, binning, camera_constant_mmA):
621        '''
622        Calculate final calibration constants
623        from known CC and camera_pixel_size.
624
625        * Once we know *CC*, *camera_pixel_size*, and *binning*,
626          the calculation of the final calibration constants
627          is surprisingly easy.
628        * The full justification of the calculation
629          is given in the comments of the source code below.
630        
631        Parameters
632        ----------
633        camera_constant_mmA : float
634            The camera constant (from R*d = CL*Lambda = CC) in [mmA].
635        camera_pixel_size_um : float
636            The real dimension of one pixel of the camera (detector) in [um].
637        binning : int
638            Binning = 1,2,4,... increases pixel size (1x,2x,4x,...)
639
640        Returns
641        -------
642        R_calibration, S_calibration, q_calibration : three floats
643            * R_calibration = the camera pixel size in [mm].
644            * S_calibration = the camera pixel size in S-vector units [1/A].
645            * q_calibration = the camaera pixel size in q-vector units [1/A].
646        '''
647
648        # What do we want to calculate?
649        #  => three calibration constants = pix_size, S-units, q-units        
650        # Brief justification:
651        #  (a) Camera Equation: R*d = CL*Lambda = CC
652        #  (b) Bragg's Law: 2*d*sin(theta) = Lamda => S*d = 1 => S = 1/d
653        #  (c) Combine (a)+(b) for a known R[um] => here: R = size of 1 pixel
654        #      in general : R*d = CC => S = 1/d = R/CC
655        #      here/below : S_calibration[1/A] = R_calibration[mm] / CC [mm*A]
656        # -----
657        # Note: camera_pixel_size is influenced by binning
658        
659        # Calculate the calibration constants acc.to justification above
660        R_calibration = camera_pixel_size_um * binning / 1000
661        S_calibration = R_calibration / camera_constant_mmA
662        q_calibration = 2*np.pi * S_calibration
663
664        # Return the three calibration constants
665        return(R_calibration, S_calibration, q_calibration)
