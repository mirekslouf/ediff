ediff.pcryst

Module: ediff.pcryst

Process polycrystalline/powder diffraction patterns.

  • XRD powder diffractograms are calculated from a structure or CIF-file.
  • ELD powder diffractograms are converted from 2D images to 1D profiles.
  • Note: XRD/PXRD = (powder) X-ray diffraction; ELD = electron diffraction.

Key classes in this module:

  • ediff.pcryst.XRD_polycrystal = calculation of a theoretical powder X-ray diffraction pattern.
  • ediff.pcryst.ELD_polycrystal = processing of an experimental 2D powder electron diffration pattern.
  • ediff.pcryst.Comparison = comparison of the XRD and ELD profiles from the previous two steps/classes.

List of all classes in this module:

  • Crystal, SimuParams, PlotParams, PeakProfile = classes that define parameters for PXRDcalculation.
  • PXRDcalculation = calculation of the theoretical XRD difractogram (basic API).
  • PXD_polycrystal = calculation of the theoretical XRD diffratogram (object-oriented API).
  • ELD_polycrystal = complete processing of the experimental ELD diffractogram (object-oriented API).
  • Comparison = final comparison of the processed XRD and ELD profiles.

 127    def get_elements(self, structure):
 128        '''
 129        Get a list of all elements, which are contained in given structure.
 130
 131        Parameters
 132        ----------
 133        structure : structure object
 134            Typically, the structure objects are
 135            created by function *read_structure_from_CIF* above.
 136
 137        Returns
 138        -------
 139        list_of_elements : list
 140            List with symbols of all elements
 141            that are present in given *structure*.
 142            
 143        Technical note
 144        --------------
 145        * The elements are obtained from *structure* object.
 146        * The structure object = pymatgen.core.structure.Structure.
 147        * The list of element names si obtained by a set of tricks
 148          specific to pymatgen-structure object.
 149        * These tricks were revealed by inspecting the original
 150          pymatgen.core.structure.Structure in Spyder,
 151          by means of shortcuts Ctrl+I (help) and Ctrl+G (go to code).
 152        * Common users do not have to use this function.
 153        * The function is just used internally, when setting
 154          temperature factors of the elements contained in given *structure*.
 155        '''
 156        list_of_atoms = []
 157        for site in structure:
 158            for sp, occu in site.species.items():
 159                list_of_atoms.append(sp.symbol)
 160        list_of_elements = np.unique(list_of_atoms)
 161        return(list_of_elements)

 288    def gaussian(X,m,s):
 289        '''
 290        Gaussian function (~ profile for PXRD calculation).
 291    
 292        Parameters
 293        ----------
 294        X : numpy array
 295            X-variable of Gaussian function.
 296        m : float
 297            Mean value = the center of Gaussian function.
 298        s : float
 299            Standard deviation = the width of Gaussian function.
 300    
 301        Returns
 302        -------
 303        NumPy array
 304            The array with Y-values of the function; Y = Gaussian(X).
 305        '''
 306        return( 1/(s*np.sqrt(2*np.pi)) * np.exp(-(X-m)**2/(2*s**2)) )
 307    
 308
 309    def lorentzian(X,m,s):
 310        '''
 311        Lorentzian function (~ profile for PXRD calculation).
 312    
 313        Parameters
 314        ----------
 315        X : numpy array
 316            X-variable of Lorentzian function.
 317        m : float
 318            Mean value = the center of Lorentzian function.
 319        s : float
 320            Standard deviation = the width of Lorentzian function.
 321    
 322        Returns
 323        -------
 324        NumPy array
 325            The array with Y-values of the functin; Y = Lorenzian(X).
 326        '''
 327        return( 1/np.pi * s/((X-m)**2 + s**2) )
 328    
 329
 330    def pseudo_voigt(X,m,s,n=0.5):
 331        '''
 332        Pseudo-Voigt function (~ profile for PXRD calculation).
 333    
 334        Parameters
 335        ----------
 336        X : numpy array
 337            X-variable of pseudo-Voigt function.
 338        m : float
 339            Mean value = the center of pseudo-Voigt function.
 340        s : float
 341            Standard deviation = the width of pseudo-Voigt function.
 342        n : float, optional, the default is 0.5
 343            Ratio between Gaussian and Lorentzian components.
 344            In the very first approximation, we can set n = 0.5.
 345            For more details, see *Notes* subsection below.
 346            
 347        Returns
 348        -------
 349        NumPy array
 350            The array with Y-values of the functin; Y = Lorenzian(X).
 351    
 352        Notes
 353        -----
 354        * Pseudo-Voigt function
 355          = linear combination of Gaussian and Lorentzian function.
 356        * In the 1st approximation:
 357            * 50/50 combination => n = 0.5
 358            * sg = sigma_Gaussian
 359              defined so that sg and s yielded the same FWHM
 360        * more details in:
 361            * https://en.wikipedia.org/wiki/Voigt_profile
 362            * https://lmfit.github.io/lmfit-py/builtin_models.html
 363        '''
 364        # (1) Calculate sg
 365        # (sg = sigma_Gaussian - see docstring above for more details
 366        sg = s/np.sqrt(2*np.log(2))
 367        # (2) Calculate Pseudo-Voigt function
 368        # (Pseudo-Voigt = n * Gausian + (1-n) * Lorenzian)
 369        pseudo_voigt = \
 370            n * PeakProfiles.gaussian(X,m,sg) \
 371            + (1-n) * PeakProfiles.lorentzian(X,m,s)
 372        # (3) Return calculated function    
 373        return(pseudo_voigt)

 469    def _calculate_diffractions(self):
 470        # Private method.
 471        # During the initialization,
 472        #  this method calculates diffractions by means of PyMatGen.
 473        # The calculated diffractions
 474        #  are saved in self.diffractions
 475        #  and used in the next step of the initialization
 476        #  to calculate diffraction profile, which is saved in self.profile.
 477        # -----
 478        
 479        # (1) Calculate intensities
 480        calculation = pmXRDCalculator(
 481            wavelength = self.simu_parameters.wavelength,
 482            debye_waller_factors=(self.crystal.temp_factors))
 483        diffractions = calculation.get_pattern(
 484            self.crystal.structure,
 485            two_theta_range = self.simu_parameters.two_theta_range)
 486        
 487        # (2) Transform diffractions: PyMatGen structure => pd.DataFrame
 488        diffractions_df = self._diffractions_to_dframe(diffractions)
 489        
 490        # (3) Return calculated diffractions
 491        # (pandas.DataFrame with cols: TwoTheta, h,k,l, dhkl, S, q, Intensity
 492        return(diffractions_df)

 495    def _calculate_diffraction_profile(self):
 496        # Private method.
 497        # During the initialization,
 498        #  this method calculates diffraction profile from self.diffractions.
 499        # The calculation does not need PyMatGen any more:
 500        #  - PyMatGen is employed only in calculation of self.diffractions
 501        #  - the self.profile is calculated by *convolution*
 502        #    of self.diffractions and self.peak.profile_type
 503        # -----
 504        
 505        # The method calculates 
 506        # (1) Base diffraction profile
 507        # = diffractogram with zero intensities
 508        # = interval theta_min to theta_max with step=0.001, zero intensities
 509        th1,th2 = th1,th2 = self.simu_parameters.two_theta_range
 510        numpoints = (th2-th1) * 100 + 1
 511        diffractogram_2theta = np.linspace(th1, th2, numpoints, endpoint=True)
 512        diffractogram_intensity = np.zeros(len(diffractogram_2theta))
 513        df = pd.DataFrame(
 514            np.transpose([diffractogram_2theta, diffractogram_intensity]),
 515            columns=['TwoTheta','Intensity'])
 516        
 517        # (2) Line diffraction pattern
 518        # = diffractogram with line intensities
 519        # (2a) read diffractions, rounded to 3 dec.places ~ step 0.001
 520        diffractions = np.round(
 521            np.array(self.diffractions[['TwoTheta','Ihkl']]),2)
 522        # (2b) insert line diffractions into zero profile from step (1)
 523        for two_theta, intensity in diffractions:
 524            # calculate index = position of the diffraction in profile array
 525            index = int(round((two_theta - th1) * 100))
 526            # insert the diffraction intensity at calculated position
 527            df.Intensity[index] = intensity
 528        
 529        # (3) Simulated diffraction profile
 530        # = diffractogram with intensity profiles
 531        # (3a) Is peak_profile_type defined?
 532        # (Argument {peak_profile_type} has default value - pseudo_voight
 533        # (Therefore, it should be defined unless user specifies None value
 534        if self.simu_parameters.peak_profile == None:
 535            print('Error: peak_profile_type is not defined!')
 536            sys.exit()
 537        # (3b) The peak_peak_profile_type is defined => calculate difractogram
 538        # Precalculate peak profile
 539        X = np.linspace(-10,10,2001,endpoint=True)
 540        peak_profile = self.simu_parameters.peak_profile(
 541            X, 0, self.simu_parameters.peak_profile_sigma)    
 542        peak_profile = peak_profile/np.max(peak_profile)
 543        # Convolve Intensities with PeakProfile
 544        df.Intensity = spConvolve(df.Intensity,peak_profile, mode='same')
 545        # Normalize the calculated intensities
 546        # (The normalization is not strictly necessary,
 547        # (as spConvolve defined above keeps the original normalization,
 548        # (but the normalization to 1 is convenient for further processing.
 549        df.Intensity = df.Intensity/np.max(df.Intensity)
 550        
 551        # (4) Add columns with diffraction vectors (S, q)
 552        # (the calculated profile has two columns: TwoTheta, Intensity
 553        # (now we insert two additional cols to get: TwoTheta, S, q, Intensity
 554        # ---
 555        # Prepare wavelenght (just shortcut for convenience)
 556        wavelength = self.simu_parameters.wavelength
 557        # Insert a column with the calculated values of diffration vector S
 558        df.insert(loc = 1, column = 'S',
 559            value = 2*np.sin(df.TwoTheta/2*np.pi/180) / wavelength)
 560        # Insert a column with the calculated values of difraction vector q
 561        df.insert(loc = 2, column = 'q', 
 562            value = 2*np.pi * df.S)
 563
 564        # (5) Return diffractogram = the whole diffraction profile
 565        return(df)

 569    def _diffractions_to_dframe(intensities):
 570        # Private method.
 571        #  Converts the calculated PyMatGen structure (intensities)
 572        #  to standard DataFrame (which is then saved in self.diffractions).
 573        # -----
 574        
 575        # Prepare hkl indexes
 576        h = []; k = []; l = []
 577        
 578        # Test if the structure is hexagonal
 579        hexagonal = (len(intensities.hkls[0][0]['hkl']) == 4)
 580        # ...and if the structure is hexagonal, prepare i-index (hkl) -> (hkil)
 581        if hexagonal: i = []
 582        
 583        # Fill in hkl (or hkil) indexes
 584        for ints in intensities.hkls:
 585            h.append(ints[0]['hkl'][0])
 586            k.append(ints[0]['hkl'][1])
 587            if not(hexagonal):
 588                l.append(ints[0]['hkl'][2])
 589            else:    
 590                i.append(ints[0]['hkl'][2])
 591                l.append(ints[0]['hkl'][3])
 592        
 593        # Create DataFrame from all values
 594        # (again, we have to consider hexagonal structures => hkil indexes
 595        if not(hexagonal):
 596            df = pd.DataFrame( 
 597                np.transpose(
 598                    [intensities.x,
 599                     h, k, l, intensities.d_hkls, intensities.y]),
 600                columns=['TwoTheta','h','k','l','dhkl','Ihkl'])
 601            df.insert(loc=5, column='S', value=1/df.dhkl)
 602            df.insert(loc=6, column='q', value=2*np.pi*df.S)
 603        else:
 604            df = pd.DataFrame( 
 605                np.transpose(

 496    def _calculate_diffraction_profile(self):
 497        # Private method.
 498        # During the initialization,
 499        #  this method calculates diffraction profile from self.diffractions.
 500        # The calculation does not need PyMatGen any more:
 501        #  - PyMatGen is employed only in calculation of self.diffractions

 503        #    of self.diffractions and self.peak.profile_type
 504        # -----
 505        
 506        # The method calculates 
 507        # (1) Base diffraction profile
 508        # = diffractogram with zero intensities
 509        # = interval theta_min to theta_max with step=0.001, zero intensities
 510        th1,th2 = th1,th2 = self.simu_parameters.two_theta_range
 511        numpoints = (th2-th1) * 100 + 1
 512        diffractogram_2theta = np.linspace(th1, th2, numpoints, endpoint=True)
 513        diffractogram_intensity = np.zeros(len(diffractogram_2theta))
 514        df = pd.DataFrame(
 515            np.transpose([diffractogram_2theta, diffractogram_intensity]),
 516            columns=['TwoTheta','Intensity'])
 517        
 518        # (2) Line diffraction pattern
 519        # = diffractogram with line intensities
 520        # (2a) read diffractions, rounded to 3 dec.places ~ step 0.001
 521        diffractions = np.round(
 522            np.array(self.diffractions[['TwoTheta','Ihkl']]),2)
 523        # (2b) insert line diffractions into zero profile from step (1)
 524        for two_theta, intensity in diffractions:
 525            # calculate index = position of the diffraction in profile array
 526            index = int(round((two_theta - th1) * 100))
 527            # insert the diffraction intensity at calculated position
 528            df.Intensity[index] = intensity
 529        
 530        # (3) Simulated diffraction profile
 531        # = diffractogram with intensity profiles
 532        # (3a) Is peak_profile_type defined?
 533        # (Argument {peak_profile_type} has default value - pseudo_voight
 534        # (Therefore, it should be defined unless user specifies None value
 535        if self.simu_parameters.peak_profile == None:
 536            print('Error: peak_profile_type is not defined!')
 537            sys.exit()
 538        # (3b) The peak_peak_profile_type is defined => calculate difractogram
 539        # Precalculate peak profile
 540        X = np.linspace(-10,10,2001,endpoint=True)
 541        peak_profile = self.simu_parameters.peak_profile(
 542            X, 0, self.simu_parameters.peak_profile_sigma)    
 543        peak_profile = peak_profile/np.max(peak_profile)
 544        # Convolve Intensities with PeakProfile
 545        df.Intensity = spConvolve(df.Intensity,peak_profile, mode='same')
 546        # Normalize the calculated intensities
 547        # (The normalization is not strictly necessary,
 548        # (as spConvolve defined above keeps the original normalization,
 549        # (but the normalization to 1 is convenient for further processing.
 550        df.Intensity = df.Intensity/np.max(df.Intensity)

 553        # (the calculated profile has two columns: TwoTheta, Intensity
