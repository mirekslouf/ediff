EDIFF :: Process powder electron diffraction pattern¶

  • Sample:
    • Au nanoislands
    • Local backup: D:\DATA.SH\STEMDIFF\X_TESTING\1AU
  • Input:
    • CIF file for calculation of theoretical PXRD diffraction pattern.
    • Image file containing experimental 2D powder electron diffraction pattern.
  • Output:
    • Several plots and text files in current dir = the directory with this notebook.
    • The final plot = comparison of experimental ED and theoretical PXRD diffractogram.

2. Theoretical PXRD¶

  • Semi-automatic step; you define I/O files in (a) and adjust parameters in (c).
  • We start with theoretical PXRD to have some idea how the data should look like.
  • To calculate the pattern, a CIF file (Crystallographic Information File) is needed.
  • CIF files for known crystal structures can be downloaded from crystallographic databases.
  • Good sources of CIF files = Crystallographic Open Database: http://www.crystallography.net/cod
In [3]:
# (a) Define input and output files
# (CIF_FILE = crystal information file, input text file from www
# (XRD_FILE = output file, calculated below, intensity = f(diff.angle/vector)
CIF_FILE = r'./CIF/au_9008463.cif'
XRD_FILE = r'./xrd_au.txt'
In [4]:
# (b) Adjust plot parameters - for PXRD plots
ediff.io.set_plot_parameters(size=(10,5), dpi=120, fontsize=8)
In [5]:
# (c) Prepare parameters for calculation
# (We simulate a fictive PXRD experiment on the crystal from CIF_FILE.
# (XTAL = crystal parameters
# (EPAR = experimental parameters
# (PPAR = plotting parameters = data for graphs
XTAL = ediff.pxrd.Crystal(structure=CIF_FILE, temp_factors=0.8)
EPAR = ediff.pxrd.Experiment(wavelength=0.71, two_theta_range=(2,140))
PPAR = ediff.pxrd.PlotParameters(x_axis='q', xlim=(1,10))
In [6]:
# (d) Calculate the PXRD diffractogram
PXRD = ediff.pxrd.PXRDcalculation(XTAL, EPAR, PPAR, peak_profile_sigma=0.1)
In [7]:
# (e) Save the PXRD calculation results
PXRD.save_diffractogram(XRD_FILE)
PXRD.save_diffractions(XRD_FILE + '.diff')
In [8]:
# (f) Plot the PXRD calculation results
PXRD.plot_diffractogram(XRD_FILE + '.png')

3. Experimental ED¶

  • Section 3.1: Read experimental 2D powder electron diffraction pattern.
  • Section 3.2: Convert the 2D-pattern to radially averged 1D profile.
  • Section 3.3: Subtract backround from the 1D profile.
  • Section 3.4: Calibrate 1D profile (convert distances in pixels to diffraction vectors q in 1/A).

4. Final comparison of ED and PXRD¶

  • Semi-automatic step:
    • Define final output file in (a).
    • Define fine_tuning constant in (b) - start with the value of 1.
    • Plot the result in (c) and see if the peak overlap - if not adjust constant in (b), re-run (c) ...
    • Once the graph in (c) is Ok, run cell (d) to save final ED profile as text for further processing or plotting.
